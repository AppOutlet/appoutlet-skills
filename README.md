# appoutlet-skills

Skills for the AppOutlet applications. This repository contains specialized skills for Gemini CLI to enhance its capabilities within the AppOutlet organization.

## Available Skills

- **[delegate-to-jules](./skills/delegate-to-jules)**: Enables delegating plans, tasks, or implementations to Jules, Google's autonomous AI coding agent. It allows for remote execution of tasks, freeing up the local environment.
- **[skill-creator](./skills/skill-creator)**: A comprehensive tool for creating, modifying, and optimizing Gemini CLI skills. It includes workflows for drafting instructions, running evaluations, benchmarking performance, and optimizing skill descriptions for better triggering accuracy.
- **[ticket-creator](./skills/ticket-creator)**: Streamlines the process of creating high-quality GitHub issues. It handles de-duplication checks, codebase investigation for context, and iterative drafting of bugs, tasks, spikes, and features using standardized templates.
